Massive Multiplayer Dome Game

I made my bachelor project toghether with 7 other students. A system for communication between mobile phones and applications for very large displays using multiple projectors, dome theaters and planetarioms in particular. Primarily for playing games letting visiting crowds in dome theaters interact with what is displayed on the screen. Server part developed in Java and used WebSockets to communicate with mobile phones. A "proof of concept" also developed, in C ++, which used the ASCII strings via TCP for communication between the server and application. Up to a hundred people could play against / with each other simultaneously on a single large display.
